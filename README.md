# lovemeal

## 安装依赖

```
yarn install
```

### 本地开发测试运行命令

```
yarn serve
```

### Compiles and minifies for production

```
yarn build
```

### 运行到微信小程序(本地调试)
```
yarn dev:mp-weixin
```

### 打包微信小程序（用于发布）
```
yarn build:mp-weixin
```
